# Chemistry

This is a gem that allows you to programmatically interact with chemical
compounds and equations.

## Installation

Add this line to your application's Gemfile:

    gem 'chemistry'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chemistry
